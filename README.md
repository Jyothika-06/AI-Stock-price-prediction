# 📈 AI Stock Price Prediction using Time Series Analysis

## 📌 Project Overview
This project focuses on predicting stock prices using Machine Learning and Time Series Analysis techniques.  
The model is trained on historical stock data to forecast future price trends.

---

## 🎯 Objective
- Analyze historical stock data
- Perform data preprocessing and visualization
- Build a predictive ML model
- Evaluate model performance

---

## 📁 Dataset
- Dataset: Apple Stock Price Data
- Features:
  - Open
  - High
  - Low
  - Close
  - Adjusted Close
  - Volume

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook / Google Colab

---

## ⚙️ Project Workflow

### 1️⃣ Data Collection
- Uploaded dataset using Google Colab

### 2️⃣ Data Exploration
- Checked shape, columns, and statistics
- Visualized data patterns

### 3️⃣ Data Preprocessing
- Handled missing values (Forward Fill)
- Removed duplicates
- One-hot encoding
- Outlier removal using IQR
- Feature scaling (MinMaxScaler, StandardScaler)

### 4️⃣ Data Visualization
- Box Plot
- Histogram
- Correlation Heatmap
- Scatter Plot

### 5️⃣ Model Building
- Algorithm: Linear Regression
- Train-Test Split (80:20)

### 6️⃣ Model Evaluation
- MAE (Mean Absolute Error)
- MSE (Mean Squared Error)
- RMSE
- R² Score

---

## 📊 Results

- ✅ R² Score: 1.0  
- ✅ Very low error values  
- Model shows perfect prediction (due to small dataset)
